# Deploy WiFi Activity Radar Backend

## Quick Deploy on Replit (vom Handy aus!)

1. Gehe zu **replit.com** auf deinem Handy
2. Drücke oben rechts auf "Sign Up" (oder "Log In" falls du schon einen Account hast)
3. Nach Login: Drücke auf "Create" (+ Symbol oben)
4. Wähle "Import from GitHub"
5. Füge die URL ein: `https://github.com/gonz4les-wq/radar`
6. Drücke "Import from GitHub"
7. Warte bis es lädt, dann drücke das Play-Button (▶)
8. Replit zeigt dir oben rechts die **Public URL** (z.B. `https://radar-xyz.replit.dev`)

## Backend-URL in der App verwenden

1. Öffne https://gonz4les-wq.github.io/Radar/ auf deinem Handy
2. Öffne die Browser-Konsole (oder notiere die Replit URL)
3. Die App verbindet sich mit dem Backend über den Parameter: `?ws=wss://radar-xyz.replit.dev/ws`

**Beispiel vollständig:**
```
https://gonz4les-wq.github.io/Radar/?ws=wss://deine-replit-url.replit.dev/ws
```

## Fertig! 🎉

Die App lädt jetzt das Frontend von GitHub Pages und verbindet sich mit dem Backend auf Replit. Alles vom Handy, alles online.
